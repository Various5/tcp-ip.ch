The Protocol Stack

Interactive TCP/IP Model with Protocol Deep Dive

7
Application Layer
Network services directly to end-user applications and processes
🖥️
Key Protocols:
HTTP/HTTPS FTP/SFTP SMTP DNS DHCP SSH Telnet SNMP
Function: Provides network services to user applications like web browsers, email clients, and file transfer programs. This is where users directly interact with network protocols and where most business logic resides.
6
Presentation Layer
Data translation, encryption, compression, and format conversion
🔒
Key Technologies:
SSL/TLS JPEG MPEG GIF ASCII MIME Compression Encryption
Function: Transforms data between application and network formats. Handles encryption/decryption, compression/decompression, and character encoding to ensure data compatibility across different systems and platforms.
5
Session Layer
Session establishment, management, and termination between applications
🔗
Session Protocols:
NetBIOS RPC SQL NFS SMB PPTP SOCKS LDAP
Function: Manages communication sessions between applications. Controls dialogue, authentication, session recovery, and maintains reliable connections across network interruptions.
4
Transport Layer
End-to-end data delivery, error recovery, and flow control
🚚
Transport Protocols:
TCP UDP SCTP DCCP QUIC SPX
Function: Ensures reliable data transmission with error checking, flow control, and segmentation. TCP provides guaranteed delivery while UDP offers speed for real-time applications like gaming and streaming.
3
Network Layer
Routing, logical addressing, and path determination across networks
🌐
Network Protocols:
IPv4 IPv6 ICMP ARP OSPF BGP RIP EIGRP
Function: Handles packet routing between networks using IP addresses. Determines optimal paths through complex network topologies and enables global internet connectivity.
2
Data Link Layer
Node-to-node delivery, frame formatting, and error detection
🔗
Data Link Technologies:
Ethernet Wi-Fi 802.11 PPP Frame Relay ATM HDLC Bluetooth Token Ring
Function: Provides reliable frame transmission over physical links. Handles MAC addressing, error detection, and flow control between directly connected network nodes.
1
Physical Layer
Raw bit transmission over physical transmission medium
⚡
Physical Media & Standards:
Copper Cable Fiber Optic Radio Waves Microwave Satellite Infrared USB Thunderbolt
Function: Defines electrical, mechanical, and timing specifications for transmitting raw bits. The actual hardware foundation that carries all digital communication across various transmission media.
