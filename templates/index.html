The Foundation of Internet Communication

Application Layer
7

HTTP/HTTPS, FTP, SMTP, DNS, DHCP - Where users interact with network services and applications communicate.

Examples: Web browsers, email clients, file transfer applications
Presentation Layer
6

Data encryption, compression, and format conversion. Ensures data is readable by the receiving application.

Examples: SSL/TLS encryption, JPEG compression, ASCII/Unicode encoding
Session Layer
5

Manages communication sessions between applications. Controls connections, sessions, and authentication.

Examples: SQL sessions, RPC calls, NetBIOS sessions
Transport Layer
4

TCP/UDP - Ensures reliable data transfer with error checking, flow control, and segmentation.

TCP: Reliable, connection-oriented | UDP: Fast, connectionless
Network Layer
3

IP, ICMP, ARP - Handles routing, addressing, and path determination across interconnected networks.

IPv4: 4.3B addresses | IPv6: 340 undecillion addresses
Data Link Layer
2

Ethernet, Wi-Fi, PPP - Provides node-to-node data transfer and error detection within a network segment.

MAC addresses, frame formatting, error detection
Physical Layer
1

Cables, radio waves, optical fibers - The actual hardware that transmits raw bit streams.

Electrical signals, fiber optics, wireless radio frequencies
